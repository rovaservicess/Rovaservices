Política de Privacidade Rova Services

Última atualização: 01/01/2024

1. Coleta de Informações

Coletamos dados para:

  • Cadastro de clientes e parceiros
  • Execução de serviços contratados
  • Comunicação comercial
  • Melhoria de nossos serviços

2. Tipos de Dados Coletados

  • Dados cadastrais (nome, CNPJ, telefone, e-mail)
  • Dados técnicos de equipamentos
  • Dados de localização para serviços
  • Registros de manutenção

3. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Parceiros técnicos para execução de serviços
  • Autoridades reguladoras quando exigido por lei
  • Sistemas de pagamento para transações

4. Direitos do Titular

Você tem direito a:

  • Acesso aos dados armazenados
  • Retificação de informações
  • Exclusão de dados pessoais
  • Revogar consentimento

5. Segurança da Informação

Utilizamos medidas como:

  • Criptografia de dados
  • Controle de acesso restrito
  • Backups periódicos
  • Treinamento de equipe

6. Cookies e Tecnologias Similares

Nosso site utiliza cookies para:

  • Funcionamento básico do site
  • Análise de tráfego (Google Analytics)
  • Personalização de conteúdo

7. Alterações na Política

Atualizações serão comunicadas através de:

  • Notificação em nosso site
  • E-mail cadastrado
  • Avisos em materiais comerciais
